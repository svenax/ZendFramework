Zend_Application Quick Start

Introdução

Zend_Application fornece um mecanismo de inicialização de aplicações que fornece recursos reutilizáveis, classes de inicialização modulares e verificação de dependência. Ele também cuida da configuração do ambiente PHP e introduz por padrão o auto-carregamento.
