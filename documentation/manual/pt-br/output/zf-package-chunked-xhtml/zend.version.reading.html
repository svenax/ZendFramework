Obtendo a Versão do Zend Framework

Zend_Version fornece a constante de classe Zend_Version::VERSION que contém uma string que identifica o número da versão de sua instalação do Zend Framework. Zend_Version::VERSION pode conter "1.7.4", por exemplo.

O método estático Zend_Version::compareVersion($version) é baseado na função » version_compare() do PHP. Esse método retorna -1 se a versão especificada é mais antiga que a versão do Zend Framework instalado, 0 se forem iguais e 1 se a versão especificada é mais recente que a versão instalada.

Example #1 Exemplo do método compareVersion()

  1. // retorna -1, 0 ou 1
  2. $cmp = Zend_Version::compareVersion('2.0.0');
